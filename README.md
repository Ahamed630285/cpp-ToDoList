# C++ To-Do List App

A simple console-based task manager written in C++.  
You can add, view, complete, and delete tasks from the list.

## Features
- Add tasks
- View tasks
- Mark tasks as done
- Delete tasks

## Technologies
- C++
- Code::Blocks IDE

## Run it:
1. Open Code::Blocks
2. Paste the code into a new project
3. Compile and run
